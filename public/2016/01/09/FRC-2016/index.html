FRC 2016 Game

We were told that this year, FIRST’s game design commitee got some help from Disney’s Imagineers to design the 2016 game.
What they came up with was… interesting, and probably the most UNIQUE game First has come out with.

And in typical disney fashion…




We get to put a foam ball in a goal, only this time, we get to pretend it’s a BOULDER.




It’s not a teeter totter, it’s a cheval de frise, there is a difference (trust us).




Do those platforms look familiar to anyone?




That one mentor who remembers the last time we built a 16 inch robot.




Field reconfigurablity isn’t quite new either, but much more complicated




Same old end game, except now we climb for warrior’s honor.

Well, what more do you expect from a game which was actually based on Monty Python and the Holy Grail.

Which reminds me, you can write your robot code in python thanks to robotPY, a python3 port of WPILIB. I promise it is worth a look.
